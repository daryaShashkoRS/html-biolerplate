The main points for score your project

  1. The whole page is done
  2. Not failure the deadline
  3. Browser Support: Google Chrome, Mozilla Firefox, Microsoft Edge.
  4. Successful per-pixel comparison via Pixel Perfect extension
  5. Use flex-box and grid
  6. Responsive for at least 3 screen sizes
  7. Header:
    • Interactive nav.
    • Think of where h1 should be used.
    • Gradients could be applied approximately. No need to make them 100% identical
    • Buttons and icons should be positioned accurately on desktop version.
  8. Main:
    • Thin lines must not be images. Think of html elements and css properties to use.
    • Care about icons, fonts and colors.
  9. Footer:
    • Email addresses must be clickable as 'mailto'.
    • Care about breadcrumbs.
  1. Not use !import in css
  2. Not use id in html
  3. БЭМ for naming classes (without translit)
  4. No nesting of selectors more than 3 levels
  5. Normolize.css should be used
  6. Web sefa fonts after the main font
  7. All states for elements should be styled
  8. Property order in scc - http://codeguide.academy/html-css.html (or your custom)
